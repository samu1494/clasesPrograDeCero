## 🏆 Práctica y Competencia

La mejor forma de aprender a programar es escribiendo código. Te dejamos el enlace a nuestra plataforma de práctica para esta sesión:

💻 **[Ir a la Competencia del Día 1](https://vjudge.net/contest/809287)**

> **💡 Consejo para la práctica:**
> No es necesario resolver todo, solo hazlo hasta donde puedas. Lo importante es practicar. Empieza por el **Problema A**, luego sigue con el **Problema B** y así sucesivamente hasta donde llegues. ¡No te rindas!
